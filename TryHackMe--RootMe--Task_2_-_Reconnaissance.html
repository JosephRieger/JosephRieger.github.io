Task 2 - Reconnaissance


Scan the machine, how many ports are open?

sudo nmap -A <IP>

Answer: 2


What version of Apache is running?

Answer: 2.4.29


What service is running on port 22?

Answer: SSH


Find directories on the web server using the GoBuster tool.


What is the hidden directory?

gobuster dir -u <IP> -w /usr/share/wordlists/dirbuster/directory-list-2.3-small.txt

This reveals a couple of directories. One is called /panel/ that allows you to upload files and another is called /uploads/, which is presumably the location where the uploaded files are stored.

Answer: /panel/
