What is the content of the flag5.txt file?

The flag is once again located in /home/ubuntu but we do not have access to it.
There is however a cronjob running under root that executes a shell script in our home directory.

We can append the following lines to it:
cp /bin/bash /home/karen/rootbash
chmod +xs /home/karen/rootbash

It is important, however, to also make the file executable. Otherwise the cronjob will not run.

Afterwards we can start this shell with ‘./rootbash -p’ to get root privileges within it.
Then we can read the contents of the file.

Answer: THM-383000283
