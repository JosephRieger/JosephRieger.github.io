Question #1: Name the directory on the FTP server that has data accessible by the "anonymous" user
1. ftp 10.10.173
2. login as “anonymous”
3. ls shows us all the directories
4. As an anonymous user we only have full permissions for the /public folder
Answer: public


Question #2: What script gets executed within this directory?
1. cd public
2. ls
3. The public folder contains one shell script named backup.sh
Answer: backup.sh
