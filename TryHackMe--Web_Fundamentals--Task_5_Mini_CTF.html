There's a web server running on http://10.10.76.191:8081. Connect to it and get the flags!


• GET request. Make a GET request to the web server with path /ctf/get
• POST request. Make a POST request with the body "flag_please" to /ctf/post
• Get a cookie. Make a GET request to /ctf/getcookie and check the cookie the server gives you
• Set a cookie. Set a cookie with name "flagpls" and value "flagpls" in your devtools (or with curl!) and make a GET request to /ctf/sendcookie


What's the GET flag?
 curl 10.10.76.191:8081/ctf/get

Answer: thm{162520bec925bd7979e9ae65a725f99f}


What's the POST flag?
 curl 10.10.76.191:8081/ctf/post -X POST --data flag_please

Answer: thm{3517c902e22def9c6e09b99a9040ba09}


What's the "Get a cookie" flag?
 https://curl.se/docs/http-cookies.html
curl 10.10.76.191:8081/ctf/getcookie -c getcookie.txt
This saves the contents of the cookie in getcookie.txt

Answer: thm{91b1ac2606f36b935f465558213d7ebd}


What's the "Set a cookie" flag?
Content of the cookie:
10.10.76.191 FALSE / FALSE 0 flagpls flagpls


The second to last parameter is the name of the cookie and the last parameter is its value. Each parameter has to be seperated by a tab and the line must end with a newline.

curl 10.10.76.191:8081/ctf/sendcookie -b flagpls
